# Portfolio Website - Praanav Bhowmik

A modern, responsive portfolio website showcasing my work as a Software Engineer & AI Innovator.

## 🌟 Features

- **Modern Design**: Beautiful gradient-based UI with smooth animations
- **Fully Responsive**: Works seamlessly on desktop, tablet, and mobile devices
- **Interactive Elements**: Smooth scrolling, hover effects, and parallax animations
- **Project Showcase**: Highlights my key projects including VisualVerse and Stock Research Agentic System
- **Skills Section**: Displays technical expertise and capabilities
- **Contact Section**: Easy ways to get in touch

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local web server (optional, for development)

### Installation

1. Clone or download this repository
2. Open `index.html` in your web browser

### Local Development

For the best development experience, use a local web server:

#### Using Python (if installed):
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

#### Using Node.js (if installed):
```bash
npx http-server
```

#### Using VS Code:
Install the "Live Server" extension and right-click on `index.html` → "Open with Live Server"

Then navigate to `http://localhost:8000` (or the port shown)

## 🎨 Customization

### Update Personal Information

1. **Name and Title**: Edit the hero section in `index.html`
2. **About Section**: Modify the about text in `index.html`
3. **Projects**: Update project cards with your actual projects
4. **Skills**: Adjust skills in the skills section
5. **Contact**: Update email and social links

### Styling

- Colors can be customized in `styles.css` under `:root` variables
- Gradients and animations can be adjusted in the CSS file
- Fonts are loaded from Google Fonts and can be changed in `index.html`

### GitHub Links

Make sure to update all GitHub repository links to point to your actual repositories.

## 🌐 Deployment

### GitHub Pages

1. Push your code to a GitHub repository
2. Go to Settings → Pages
3. Select your branch (usually `main` or `master`)
4. Your site will be available at `https://yourusername.github.io/portfolio`

### Netlify

1. Drag and drop your project folder to [Netlify Drop](https://app.netlify.com/drop)
2. Your site will be live instantly!

### Vercel

1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in your project directory
3. Follow the prompts
